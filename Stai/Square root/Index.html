To start the simulator, press the Start button. The simulator is performed for a while, without the help of a calculator, 6 seconds are given for each answer. You must correctly answer all 64 combinations that are issued randomly.

In case of at least one error or expiration of the response time, the test is not passed. To successfully pass the test, you must answer all 64 combinations correctly! The response field is cleared automatically after the response time. You can enter the answer from the computer keyboard or using the additional green buttons when using a mobile device or tablet.
